static void Antura.UI.MinigamesUI.Init ( MinigamesUIElement  _elements)
inlinestatic

Instantiates the Minigames UI in a scene, with the given parameters. The minigames UI is destroyed when a new scene is loaded.

NOTE: calling this method when a MinigamesUI is already in the scene will destroy the current one and create a new one.

NOTE: after calling this method, you will have to call the Setup method on each separate element (except for Starbar that doesn't need it).

Parameters
_elementsUI elements, uses Flag enum. For example, to activate both Starbar and Timer, use:
Init(MinigamesUIElement.Starbar | MinigamesUIElement.Lives);

while to activate only the Timer, call:

Init(MinigamesUIElement.Timer);
