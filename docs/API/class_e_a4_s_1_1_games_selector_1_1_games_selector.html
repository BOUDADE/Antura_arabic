Detailed Description

Handles the GamesSelector logic, showing bubbles for all mini-games that should be played next.

This can be instantiated in any scene. To create and start it automatically, just call the static Show method. When the user has opened all bubbles and the app should move on, the OnComplete event will be dispatched.
